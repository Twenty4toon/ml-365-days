# ML Engineer - 365 Days Roadmap

A self-paced roadmap to become an ML Engineer in 365 days. From Python basics to deploying production AI products.

## Quick Start

### Local (Node.js server)
```bash
npm install
node server.js
```
Visit `http://localhost:3000`

### Static (GitHub Pages)
1. Create a new repo on GitHub
2. Push these files:
   - `index.html` (frontend)
   - `README.md`
3. Go to **Settings → Pages**
4. Set Source to **main branch**
5. Your site will be live at `https://yourusername.github.io/repo-name/`

## Project Structure

```
ml-roadmap-365-daily/
├── index.html      # Static frontend (works on GitHub Pages)
├── server.js       # Node.js API server (for dynamic backend)
```

## Roadmap Overview

| Phase | Days | Focus |
|-------|------|-------|
| M1–2 | 1–56 | Python + Math (NumPy, Pandas, Linear Algebra, Calculus, Stats) |
| M3–4 | 57–112 | ML Fundamentals (Supervised, Unsupervised, Model Eval, Feature Engineering) |
| M5–6 | 113–168 | Deep Learning (PyTorch, CNNs, RNNs, Transformers) |
| M7–8 | 169–224 | NLP + HuggingFace (BERT, LoRA, RAG Systems) |
| M9–10 | 225–280 | MLOps + Backend (FastAPI, Docker, AWS, Monitoring) |
| M11–12 | 281–365 | Build + Ship (3 AI Products, Portfolio, Job Hunt) |

## Daily Schedule

- **6:00–7:30am** — Core Learning (Theory, courses, videos)
- **7:30–8:30am** — Coding (Build, experiment, practice)
- **8:30–9:00am** — LinkedIn/GitHub (Post progress, commit code)
- **Evening (opt.)** — Reading (Papers, docs, review)
